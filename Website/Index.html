Basic HTML

To make a HTML-page, you need to learn the basics. Making a HTML-page is not always easy, but that's why I'm teaching you this. Let's start with the <html> element. This container is the whole document, where you put your CSS, Javascript and all tags. Inside those you put the <head> element which is used to put all your metadata (data about data).

Now it should look like this:

<html> <head> </head> </html>

Inside the <head> element, you put the <title> element. The <title> defines the title of the document.

  • defines a title in the browser tab.
  • provides a title for the page when it is added to favorites.
  • displays a title for the page in search engine results.

    • How the code should look now:

    <html> <head> <title>Casper's HTML Guide</title> </head> </html>

    Okay. Now we're done with the <head> element - for now.

    After the </head> you make the <body> element. Inside this container you put the HTML tags. Something like this:

    <body> <h1>This is a Header</h1> <p>This is a Paragraph</p> </body>
